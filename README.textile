*+Description+*
*upscrn_cli* is a multi-platform dekstop client for upscrn (url upscrn). It gives the user an easy
way to capture and share desktop screenshots, by uploading it to upscrn and returning the
public url for said image.

*+How to execute+*
Just run 'jruby src/upscrn.rb' from the project folder.
